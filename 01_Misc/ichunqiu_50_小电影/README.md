## 解题思路
---

是一个Gif文件，但是提示格式错误打不开。使用notepad++打开文件，发现文件头开头是9a，然而Gif文件格式开头应该是GIF89a，修改后可以打开。打开后最后一帧图片就是flag。

![](images/ctf-2021-06-07-08-48-32.png)

使用`StegSolve`分帧查看
