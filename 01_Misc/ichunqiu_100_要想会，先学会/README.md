## 解题思路

![](images/ctf-2021-06-07-15-27-47.png)

```python
#!/usr/bin/env python2
# -*- coding: utf-8 -*-
a=[144,144,150,139,145,165,120,139,91,160,93,167,70]
for j in range(-50,50):
  flag=''
  for i in a:
    flag+=chr(i+j)
  print flag
```
按照时间排序，得到一串奇怪的东西，转ascii有很多字符不可见，猜测是有一些偏移，于是写了个py脚本暴力所以可能的偏移，得到flag
